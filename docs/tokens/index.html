    tokens

    variables

    prefix

    $prefix: null !default;

    Description

    Prefix to be applied to all CSS custom properties. Set to false or null to exclude prefix.

    Type

    String

    Example

    @use '@dwijbavisi/mood-swings/tokens' with (
    $prefix: 'mood-swings'
);

    accent

    $accent: hsl(280deg 60% 30%) !default;

    Description

    Accent color for the color scheme.

    Type

    Color

    Example

    @use '@dwijbavisi/mood-swings/tokens' with (
    $accent: hsl(36deg 100% 50%)
);

    variant

    $variant: hsl(150deg 60% 30%) !default;

    Description

    Variant color for the color scheme.

    Type

    Color

    Example

    @use '@dwijbavisi/mood-swings/tokens' with (
    $variant: #1f7a4d
);

    mood

    $mood: #000 !default;

    Description

    Mood color for the color scheme. By default, this is the same set to black. When black, it does not affect the color scheme. Otherwise, it blends with accent and variant colors.

    Type

    Color

    Example

    @use '@dwijbavisi/mood-swings/tokens' with (
    $mood: #451476
);
